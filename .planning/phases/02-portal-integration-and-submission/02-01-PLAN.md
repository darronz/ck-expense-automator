    **1. parseDateToISO(ddmmyyyy: string): string**
    Convert dd/mm/yyyy to yyyy-mm-dd. Split on '/', return `${year}-${month.padStart(2,'0')}-${day.padStart(2,'0')}`.

    **2. parseAmount(text: string): number**
    Strip non-numeric/non-dot characters from text (e.g. "£18.67" → 18.67). Use `text.replace(/[^0-9.]/g, '')` then `parseFloat(...) || 0`.

    **3. triggerPaymentTypeAndSuspenseItems(): void** (internal — not exported)
    Access `window.$` via `const win = window as any; const $ = win.$`.
    Step 1: `$('[name="ExpensePaymentTypeId"]').val('2').trigger('change')`
    Step 2: `$('[name="IsMappedToSuspenseItems"]').filter(':checkbox').prop('checked', true).trigger('change')`

    **4. waitForDataTable(timeoutMs = 5000): Promise&lt;void&gt;** (internal — not exported)
    Check immediately if `$('#DataTables_Table_1').length > 0` — if yes, resolve immediately.
    Otherwise: attach a MutationObserver on `document.body` with `{ childList: true, subtree: true }`.
    On each mutation, check if `$('#DataTables_Table_1').length > 0` — if yes, disconnect and resolve.
    Set a setTimeout for timeoutMs that disconnects the observer and rejects with `new Error('TIMEOUT: DataTable #DataTables_Table_1 did not appear within ' + timeoutMs + 'ms')`.

    **5. export async function readSuspenseItems(claimId: string): Promise&lt;SuspenseItem[]&gt;**
    This is the primary export for this plan. Full implementation:

    ```typescript
    export async function readSuspenseItems(claimId: string): Promise<SuspenseItem[]> {
      const win = window as any;
      const $ = win.$;

      // Check if DataTable is already visible (e.g. form was already in correct state)
      if (!($('#DataTables_Table_1').length > 0)) {
        triggerPaymentTypeAndSuspenseItems();
        await waitForDataTable(5000);
      }

      // If still not present after trigger, return empty (no suspense items)
      if (!($('#DataTables_Table_1').length > 0)) {
        return [];
      }

      const dt = $('#DataTables_Table_1').DataTable();
      const items: SuspenseItem[] = [];

      // Show all rows — pass false to draw() to avoid scroll reset
      dt.page.len(-1).draw(false);

      try {
        dt.rows().every(function(this: any, rowIdx: number) {
          // Deselect all rows, then select only this row to populate the hidden field
          dt.rows().deselect();
          dt.row(rowIdx).select();

          const suspenseId = ($('#MappedSuspenseItemIds').val() as string) ?? '';
          if (!suspenseId) return; // Row selection didn't populate field — skip

          const rowNode = dt.row(rowIdx).node() as HTMLTableRowElement;
          const cells = rowNode.querySelectorAll('td');
          // Cell order: checkbox(0), date(1), description(2), amount(3)
          const dateText = cells[1]?.textContent?.trim() ?? '';
          const description = cells[2]?.textContent?.trim() ?? '';
          const amountText = cells[3]?.textContent?.trim() ?? '';

          items.push({
            id: suspenseId,
            date: dateText,
            isoDate: parseDateToISO(dateText),
            description,
            amount: parseAmount(amountText),
          });
        });
      } finally {
        // ALWAYS clean up — even if iteration throws
        dt.rows().deselect();
        ($('#MappedSuspenseItemIds') as any).val('');
      }

      return items;
    }
    ```
